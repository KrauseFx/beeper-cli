# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-19
### Added
- Read-only CLI to query the local Beeper SQLite database
- Thread listing and thread detail commands
- Message browsing with sender resolution
- Full-text search using Beeper's built-in FTS index
- Search context windows around matches
- Optional DM name resolution via bridge databases
- JSON output for agent-friendly integrations
- GitHub CI workflow and linting configuration
